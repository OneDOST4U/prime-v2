# Frontend

React application structure, component map, wireframes, and screen specifications.

**Target deliverables:** UI architecture, form-rendering plan, role-based dashboard specs

**Approval required before:** Frontend implementation (Phase 5 prototype approval)

## UI Standards (Mandatory)

All UI work must follow **[UI-DESIGN-STANDARDS.md](UI-DESIGN-STANDARDS.md)**:

- Modern, responsive design for **mobile, tablet, and desktop**
- **Right-side navigation** for all authenticated users — not a top navbar
- Same nav placement for every role; menu items vary by permissions

Wireframes and prototypes must show the right-nav shell at all three breakpoints before Phase 5 approval.
